# LeetCode Improvement Coding Buddy Bot

A smart, automated coding coach that analyzes your GitHub pushes, tracks your mistake patterns, and delivers personalized weekly improvement plans — just like having a private LeetCode tutor.

---

## Overview

The **LeetCode Improvement Coding Buddy** is a GitHub-connected bot that helps you level up faster by:

- Monitoring your code commits in real time
- Analyzing where you struggle
- Adapting focus like Anki cards
- Sending you a weekly email summary with clear action items

Stop random grinding. Start deliberate, data-driven improvement.

---

## Features

- **Initial + Monthly Goal Setting** — Tell the bot what you want to improve (e.g. “graphs”) and it will ask you for a new focus area every month.
- **GitHub Repository Monitoring** — Only needs read access to one repository of your choice.
- **Automatic Code Analysis** — Every time you push code, the bot analyzes it and identifies improvement opportunities.
- **Mistake Pattern Detection** — Tracks recurring mistakes across problems (inspired by popular “mistake logs” used in LeetCode/DSA prep).
- **Anki-Style Adaptive Focus** — Automatically shifts attention to the topics you’re struggling with the most.
- **Focus Mode Toggle** — Lock the bot on one topic for a set period (e.g. 3 weeks) when you want to go deep.
- **Direct LeetCode Integration** — After analyzing your push, the bot links to relevant LeetCode problems and recommends the best next ones.
- **Weekly Email Summary** — Every Sunday you receive a clean email with:
  - Summary of problems solved
  - Personalized improvement suggestions
  - Recommended problems for the next week
  - Mistake pattern alerts
- **Quick Feedback Loop** — Rate each week’s analysis so the bot gets smarter over time.
- **Progress Dashboard** — Visual long-term insights including trends, heatmaps, streaks, and improvement metrics.

---

## How It Works

1. You connect the bot to **one GitHub repository**.
2. Every time you push LeetCode solutions, the bot analyzes the code.
3. On **Sunday**, you receive a detailed email summary.
4. The bot adapts weekly based on your performance and feedback.
5. Every month it checks in to see if you want to add a new focus area.

---

## Weekly Workflow

- **Sunday Morning** — Receive personalized email summary
- **Throughout the Week** — Push code → bot analyzes in background
- **End of Week** — Bot prepares next week’s recommendations
